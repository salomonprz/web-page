Bulk Upload Products

Select a file with your product data in JSON, CSV, or text format and upload it into the database.

File Format and Instructions

Ensure your file is in one of the following formats: JSON, CSV, or plain text.

Example JSON format:

[
    {
        "productId": "123",
        "name": "Product Name 1",
        "description": "Description here",
        "categoryId": "Category1",
        "imagePath": "/images/product1.png",
        "price": 100
    },
    {
        "productId": "124",
        "name": "Product Name 2",
        "description": "Description here",
        "categoryId": "Category2",
        "imagePath": "/images/product2.png",
        "price": 150
    }
]

For CSV and text files, ensure each line represents a single product with comma-separated values.
